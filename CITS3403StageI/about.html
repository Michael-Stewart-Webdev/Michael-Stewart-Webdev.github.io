About BIP!


Creator

BIP! was created by Michael Stewart (Student number: 20947715) as part of a project for CITS3403 at the University of Western Australia.


Optimal Screen Resolution

I've tested the site on 1280x960, 1360x768, and 1920x1080. The website should work properly on all of these resolutions. Lower resolutions may present the page less nicely than higher resolutions.

General notes about the site

Clearly the site is in a very non-useable state at the moment. The links work, but there's no underlying database. The pages available currently reflect what the final site will look like, but do not allow anything more than that.

The site is all HTML5, using CSS3. Every page has been successfully validated at The W3C Markup Validation Service.


Why isn't this site styled with Bootstrap?

I figured I'd probably learn a lot more about CSS, HTML, and JavaScript by starting the website from scratch. I used Bootstrap for my CITS2200 project last year. It worked out alright, but this time I wanted to create my own design instead of relying on stylesheets a lot of other people will probably use.

Unfortunately, a negative consequence of not using Bootstrap is that the code behind this website is more complicated than it would have been had I used it...

I guess the site looks pretty Bootstrap-ish anyway, haha. Oh well.
