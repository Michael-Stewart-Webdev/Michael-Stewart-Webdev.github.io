Game Information

Aim

You are the pilot of a powerful blue spaceship and must navigate through various nebulae.

Destroy as many aliens along the way as possible to achieve a hiscore. If you chain kills together you'll earn a combo bonus.

There's only one level so far but I will add more if I have time!

Controls

Key Action
Arrow keys Move
Space Shoot
Q Special weapon
ESC Restart game

Difficulty Levels

Difficulty Description
Easy You take half damage, and less enemies spawn.
Normal You take regular damage. The game is fairly challenging.
Hard You take double damage, everything moves faster, and many more enemies spawn. Intended to be a very hard challenge.

Enemies

Name Info
Blob A slow-moving, weak enemy.
UFO Flies around randomly on the screen, causing problems if allowed to roam.
Ghost Moves in a swaying motion. Fairly hard to kill.
Pink Ghost Charges at you very quickly when you move directly underneath it.
Awards a lot of points, but is very dangerous.
Blob Gunner Very similar to the regular Blob, but also shoots in a single direction.
UFO Gunner Moves in the same pattern as the regular UFO but also shoots at you.

Pickups

Name Info
Health Restores 20 Health.
Quick Reflexes Slows time considerably, allowing you to easily take out many enemies.

Resources Used

I developed all of the game code (except for some small functions as noted in the code), sprites, and some of the sounds.

The music, the rest of the sounds, and backgrounds, were obtained from external sources, including:

  • LF2 (some sounds)
  • Wikipedia (backgrounds)
  • MidiWorld (music)
  • Quake III Arena (player hit sound)
  • Fixedsys Excelsior (font)
  • Subtle Patterns (page background)
  • Dreamstime ("Quick Reflexes" background)

I've had some of the other sounds for a long time (more than 10 years), and no longer remember their source.

Ideal Browser

This game runs best on Google Chrome. If you experience occasional slowdowns on Firefox, please try it out on Chrome.
