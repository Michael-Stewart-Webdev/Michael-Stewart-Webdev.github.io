Technologies

Back-endRuby on Rails
HostingHeroku

Central Regional TAFE is a Western Australian TAFE that specialises in providing flexible course delivery methods.

In 2015, I expressed my interest in completely redesigning and redeveloping Central Regional TAFE's (then C.Y. O'Connor Institute's) website. I previously worked there in the marketing department for more than a year, designing a range of marketing material such as flyers, brochures and posters.

Energia MineralsX

Technologies

Front-endHTML5, CSS3, jQuery
Back-endRuby on Rails
HostingHeroku
Version controlGit, GitHub

Energia Minerals is a mining company that owns multiple mining projects in both Australia and Italy.

I was hired by Energia as a freelance web developer to completely redesign and redevelop their website in early 2015.

The site is built using Ruby on Rails, and features a number of external plugins that provide functionality such as stock prices and sharemarket document hosting. Visitors of the site may subscribe to receive updates via email.

Perhaps the most notable feature of the site is the ability to switch between English and Italian. Almost all textual content on the site has both an English and Italian translation, which was facilitated using Rails' I18N module.

NebulaX

Technologies

Front-endHTML5, Javascript
Back-endNone
HostingGitHub
Version controlGitHub

Nebula is a Javascript game that runs on HTML5 Canvas. I originally made the game using Game Maker some time in 2011, and decided to remake it using Javascript between 2015 and 2016 for the learning experience.

The player controls a blue spaceship that must defeat alien spacecraft and obtain a high score. The player must navigate through vicious asteroid fields and ever-increasing numbers of enemies.

The game has a strong focus on the interaction between the background music (David Bowie's Space Oddity, in 8-bit form) and the events taking place on the screen.

There is currently only one level, but if I have time I will add new levels in the future. Each level is designed to represent a particular nebula. The first level, for example, depicts the Little Ghost Nebula.

The game runs best on Google Chrome. Please feel free to try it out and let me know your feedback!

You can read more information about the game here.

Data Quality Assessment ToolX

Technologies

Front-endHTML5, CSS3, jQuery, Bootstrap
Back-endRuby on Rails
HostingHeroku
Version controlGit, GitHub

The Data Quality Assessment Tool allows users to input data related to their equipment and receive prognostics information in the form of radar charts. This information may be used to determine whether it is possible to make a particular decision regarding an asset.

The web application was created for Professor Melinda Hodkiewicz from The University of Western Australia and Professor Neil Montgomery from The University of Toronto.

DQAT uses Chart.js to transform the input scores of its users into presentable charts for analysis.

On the server-side of the application is a fairly complex relational database design that allows for the addition/removal of chart types, input scores and other data classes without the need to modify the application source code.

Please feel free to try out the demo of the application.

UWA Plant GuideX

Technologies

Front-endHTML5, CSS3
Back-endRuby on Rails
HostingWas hosted at UWA
Version controlGit, GitHub

In 2013 I worked with four other students to create a plant guide for the Faculty of Landscape, Architecture and Visual Arts at UWA. This was part of CITS3200 - Professional Computing.

The site allowed students to log in and search through a long list of plants found in Western Australia, and view detailed information regarding each plant. Students could also upload plants themselves. I built an interface for administrators to log in and manage the application.

One of the key features was the ability to search the entire database for plants based on a number of fields. The plants could also be sorted and viewed in either a grid or a list.

There was a strong emphasis on mobile usability, so the design of the site was heavily designed around the aesthetic of a mobile phone app. It features two different colour schemes.

Unfortunately the application is no longer hosted at UWA.

PhD Web ApplicationX

Technologies

Front-endHTML5, CSS3, d3.js
Back-endNode.js & Express, Python
HostingNectar
Version controlGit, Bitbucket

I am currently working on a web application as part of my PhD research. The application visualises the output of my Natural Language Processing system, which is written in Python. The system uses the NLTK library to perform NLP-related tasks such as lemmatization and part-of-speech tagging.

The app is written in Node.js, and is the first Node app I have created. It uses the Express web framework.

The timeline chart, pictured left, visualises each accident that occured in the mining industry in Western Australia during a one year period. This chart was programmed using D3.js. It may be aggregated by day, week, month, and year.

The application features a Word Cloud, which was also created using D3. The Word Cloud shows all noun phrases present in the entire dataset.

Due to the confidential nature of the workplace accident data, I am unable to provide a link to the application.
